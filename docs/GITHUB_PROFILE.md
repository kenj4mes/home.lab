## üîß PAT Token Generation

### Step-by-Step

1. Go to [GitHub Settings ‚Üí Developer Settings ‚Üí Personal Access Tokens ‚Üí Tokens (Classic)](https://github.com/settings/tokens)

2. Click **"Generate new token (classic)"**

3. Select scopes:
   ```
   ‚úÖ repo (Full control of private repositories)
   ‚úÖ read:user (Read all user profile data)
   ‚úÖ read:org (Read organization membership)
   ‚¨ú workflow (Optional - for self-updating workflows)
   ```

4. Copy token immediately (you won't see it again)

5. Add to repository secrets:
   - Go to `username/username` repo ‚Üí Settings ‚Üí Secrets ‚Üí Actions
   - Add `METRICS_TOKEN` with the token value
   - Add `GH_TOKEN` with the same token

## üèÜ Achieving S+ Rank

### The Algorithm Weights

The S+ rank algorithm uses weighted percentiles:

| Metric | Weight | Strategy |
|--------|--------|----------|
| **Stars** | 33.33% | Create useful repos, contribute to popular projects |
| **Pull Requests** | 25% | Contribute to open source |
| **Commits** | 16.67% | Consistent daily commits |
| **Issues** | 8.33% | Report bugs, suggest features |
| **Reviews** | 8.33% | Review others' PRs |
| **Followers** | 8.33% | Build community presence |

### Critical Parameters

```markdown
![Stats](https://github-readme-stats.vercel.app/api?username=YOUR_USERNAME&include_all_commits=true&count_private=true&show_icons=true&theme=tokyonight&hide_border=true&rank_icon=github)
```

**Key flags:**
- `include_all_commits=true` - Count ALL commits (including private)
- `count_private=true` - Include private repo contributions
- `rank_icon=github` - Clean rank display

## üê≥ Self-Hosting (Advanced)

### Docker Metrics Instance

```bash
docker run -d \
  --name github-metrics \
  -e METRICS_TOKEN=ghp_YOUR_TOKEN \
  -p 3000:3000 \
  ghcr.io/lowlighter/metrics:latest
```

### Vercel Deployment

1. Fork [lowlighter/metrics](https://github.com/lowlighter/metrics)
2. Connect to Vercel
3. Add `GH_TOKEN` environment variable
4. Access via `https://your-metrics.vercel.app/YOUR_USERNAME`
